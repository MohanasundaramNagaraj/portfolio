# Mohanasundaram N — Portfolio

Static portfolio site (HTML + Tailwind CDN + Vanilla JS + GSAP + AOS + Lucide).

## Deploy to GitHub Pages
1. Create a new repo (e.g. `portfolio`) and push these files to the root.
2. Repo → Settings → Pages → Source: `main` branch, `/ (root)` folder.
3. Site goes live at `https://<your-username>.github.io/portfolio/`.

## Files
- `index.html` — markup, SEO meta, sections
- `style.css` — custom animations, glass cards, cursor glow
- `script.js` — skills/projects data, particles, counters, GSAP/AOS init
- `assets/Mohanasundaram_N_Resume.pdf` — downloadable resume

## Customize
- Update email/GitHub/LinkedIn links in the Contact section of `index.html`.
- Edit `SKILLS` / `PROJECTS` arrays in `script.js`.
